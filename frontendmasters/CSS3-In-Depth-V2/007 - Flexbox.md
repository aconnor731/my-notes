# 007 - Flexbox

## Goal of Flexbox and Demos

```css
body > div {
    display: flex;
}
div > div {
    flex: 1;
}
```

<div style="display: flex;">
  <div style="flex: 1; border-style: dotted;">
      hello
  </div>
  <div style="flex: 1; border-style: dashed;">
    world    
  </div>
  <div style="flex: 1; border-style: dotted;">
      hello
  </div>
</div>
